# PeriaCoin ($PERIA)

> Free crypto mining on BNB Smart Chain — No hardware. No investment. Just show up.

## Overview

PeriaCoin ($PERIA) is a BEP-20 utility token deployed on BNB Smart Chain with a fixed total supply of 100,000,000 tokens. The project introduces MiningPeria, a cloud-based mobile mining platform that enables anyone worldwide to earn $PERIA tokens for free — without hardware, investment, or technical knowledge.

## Business Model

MiningPeria operates on a session-based free mining model. Users initiate mining sessions through the mobile app or web platform. The platform distributes $PERIA rewards from the locked smart contract pool.
