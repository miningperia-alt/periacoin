# PeriaCoin ($PERIA)
A free cloud-based crypto mining platform built on **BNB Smart Chain (BSC)**.

## Technology Stack
- **Blockchain**: BNB Smart Chain (BEP-20)
- **Smart Contracts**: Solidity 0.8.31, OpenZeppelin
- **Frontend**: PHP 8+ / MySQL
- **Mobile**: Android (Google Play) / iOS (App Store)
- **Security**: Firebase Auth, JWT, Cloudflare, VPN/proxy blocking

## Supported Networks
- **BNB Smart Chain Mainnet** (Chain ID: 56)

## Features
- Free cloud mining — no hardware, no investment required
- Level system with up to 3x earning multiplier
- 3% burn mechanism on every withdrawal (deflationary)
- Halving every 6 months — fixed supply of 100,000,000 PERIA
- On-chain withdrawals to any BNB Chain wallet
- Smart contract verified on BSCScan — Security Score: 93.54/100
